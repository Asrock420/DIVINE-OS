# 👑 Divine OS - Mega ERP & CRM SaaS Platform

**Divine OS** is a highly optimized, enterprise-grade, and mobile-responsive Custom ERP & CRM system. Built with a focus on high-speed data caching, role-based access, and seamless sales automation, this platform bridges the gap between digital workflows and physical corporate processes.

---

## 🚀 Key Features

* **🛡️ Role-Based Access Control (RBAC):** Dynamic permission matrix for Executive Management, Sales, Marketing, CR & Accounts, Admin, and HR.
* **📊 Real-Time KPI Dashboard:** Live tracking of Total Bookings, Revenue, Cash Flow, Pending Tasks, and Top Sales Performers.
* **📈 Sales Kanban Board (Action Wings):** A fully interactive drag-and-drop styled pipeline. Click on a lead to open "Action Wings" for instant Call Notes and automated Meeting Scheduling.
* **📝 Dynamic Bookings Module:** Manage closed deals, lead types (SGL/MPL), and update recurring down payments directly from the dashboard.
* **🧾 4-Level Requisition System:** A smart expense management tool with dynamic department-based dropdowns. Includes a 4-layer approval chain (TL ➡️ Admin ➡️ CEO ➡️ Accounts).
* **🖨️ Auto Cash Voucher Print:** Upon final approval, the system auto-generates a digitally verified, printable Cash/Payment Voucher complete with system audit trails and the Divine Group letterhead.
* **⚡ Super Fast Data Caching:** Built-in JS local caching mechanism ensuring zero-latency navigation between modules.
* **📱 100% Mobile Optimized:** Innovative "Card View" CSS ensures tables never scroll horizontally on mobile screens, providing a native app-like experience.

---

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+), Local Storage Caching.
* **Backend:** Google Apps Script (REST API).
* **Database:** Google Sheets API (Used as a highly scalable, zero-cost NoSQL alternative).
